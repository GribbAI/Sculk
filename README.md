# Sculk

> Client-side player tracking and highlighting module for Minecraft.

Sculk Tracker is a lightweight Fabric-based client mod designed for tactical player awareness in PvP.  
It provides realtime tracking and visual highlighting of nearby players without requiring server-side installation.

Releases are available via GitHub Releases or Modrinth.


---


## Features

- Realtime player entity tracking
- Direction-aware targeting logic
- Configurable filtering system
- Lightweight rendering overlay
- Minimal performance overhead
- Fully client-side (no server dependency)
